### Информация о предметной области
Данная база данных содержит две таблицы: 'user' и 'role'

Таблица 'user' содержит в себе следующие параметры:
 - iduser;
 - username;
 - password;
 - idrole.

Таблица 'role' характерезуется следующими параметрами:
 - idrole;
 - name.
